# 悟空吞天传

Q版西游题材 2D 吞噬成长游戏。操控悟空吞噬更弱的小妖和灵气，收集随机法宝，成长后挑战更强敌人。

## 玩法

- 键盘方向键或 WASD 移动，鼠标/触屏拖动也可以控制方向。
- 吞噬比自己弱的敌人会获得分数和成长值。
- 碰到更强敌人会损失生命，生命归零后可以重开。
- 随机法宝会改变局势：蟠桃、筋斗云、金箍棒、定身符、护身毫毛、紫金葫芦。
- 按 `Space` 或 `P` 暂停/继续。

## 本地运行

静态文件可直接托管。线上默认从 CDN 加载 Phaser，本地保留 `vendor/phaser.min.js` 作为兜底。推荐用本地 HTTP 服务打开：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

如果要跑稳定测试模式：

```text
http://localhost:4173/?debug=1&seed=20260530
```

## 发布

项目适合直接部署到 GitHub Pages。仓库推送后，在 GitHub Pages 中选择 `main` 分支根目录作为发布源即可。
